#### To try this project in your machine, run this command:
`$ git clone https://github.com/Drzaln/PergiTravel.git`

`$ cd PergiTravel`

`$ npm install`

`$ npm start`

#### Build project
`$ npm build`

You can also deploy this project to hosting like Netlify, Firebase, etc.
